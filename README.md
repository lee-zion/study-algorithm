# 작업 디렉토리 생성방법

```
./ec 문제제목 [문제링크=#link]
```

## 예시

```
./ec AB https://www.acmicpc.net/problem/1008
./ec 'DFS와 BFS'
./ec 'DFS와 BFS' https://www.acmicpc.net/problem/1260
```

## `./ec` 사용시 주의점

- 파일 이름 및 디렉토리에 사용할 수 없는 문자 사용 시 README 수정 안 됨
- `# % & { } \ $ ! ' " : @ ,`

# 샘플 수정 방법

- `.sample` 디렉토리 내 `README.md, main.py, input.txt, test.py` 수정

# STANDARD 문제 구분

- 문제 난이도의 기준이 되는 문제
- 문제제목 앞에 ★ 추가
